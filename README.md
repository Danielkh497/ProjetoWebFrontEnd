# 📌 Landing Page - Agendamento de Campos Society

## 📖 Sobre o Projeto
Este projeto consiste em uma landing page desenvolvida para um sistema de agendamento de campos society.

O site permite que os usuários:
- Visualizem os campos disponíveis
- Consultem horários
- Realizem agendamentos de partidas

O projeto também possui telas de login e cadastro de usuários.  
Porém, nesta versão, não há integração com banco de dados, sendo apenas a interface visual das páginas.

---

## ⚽ Funcionalidades
- 📅 Visualização de horários disponíveis
- 🏟️ Visualização dos campos
- ✅ Agendamento de partidas
- 🔐 Tela de login
- 📝 Tela de cadastro
- 📱 Interface simples e responsiva

---
