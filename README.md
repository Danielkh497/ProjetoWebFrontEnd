📌 Landing Page - Agendamento de Campos Society
Sobre o Projeto

Este projeto é uma landing page desenvolvida para um sistema de agendamento de campos society.
O site permite que os usuários visualizem os campos disponíveis, consultem horários e realizem agendamentos de forma simples e rápida.

O projeto também possui telas de login e cadastro de usuários. Porém, nesta versão, não há integração com banco de dados, sendo apenas a interface visual das páginas.

⚽ Funcionalidades
Ver horários disponíveis
Visualizar os campos
Agendar partidas
Tela de login
Tela de cadastro
Interface simples e responsiva

🛠️ Tecnologias Utilizadas
HTML
CSS
